# 🧑‍💻 Shreyas Jayan's Portfolio Website

## 📌 What’s this about?
Hey! 👋 This is my little corner on the web — a personal portfolio built using just good ol’ HTML and CSS (no fancy frameworks yet!). It's where I share what I’ve been up to as a B.Tech CSE student at Amrita Vishwa Vidyapeetham, Amritapuri.

---

## 🗂️ How it's structured

- **`index.html`**  
  Home sweet home — intro, a bit about me, and links to socials.

- **`education.html`**  
  A timeline-ish look at my schooling and where I’m studying now.

- **`project.html`**  
  Ongoing stuff I’m building (and occasionally breaking 😅). Progress notes included.

- **`skills.html`**  
  Tech stack, programming languages, and things I’m currently exploring.

- **`contact.html`**  
  Just the basics: how to reach me. Includes email, GitHub, LinkedIn, etc.

- **`styles.css`**  
  Central stylesheet — handles layout (mostly flexbox), some card-style UIs, and a sidebar.

- **`assets/`**  
  Where I’m keeping images (using a placeholder profile pic for now).

---

## 🔗 Navigation
Each page has a consistent sidebar on the left to hop between sections easily. Might add some animations later... maybe.

---

## 🖥️ Mobile-friendliness
It's responsive-ish thanks to flexbox. I haven't added full media query breakpoints yet, but it's in the backlog.

---

## 🚀 Hosting
The site is live via **GitHub Pages**.  
If you're checking it locally, just clone the repo and open `index.html` in a browser.
